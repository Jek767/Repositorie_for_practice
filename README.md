# Приложение - конвертер валют
В этом репозитории представлена программа, позволяющая проводить конвертации валют.
Приложение имеет оконный интерфейс, возможность выбора цветовой темы(темной/светлой) и языка(русского/английского).
# Цель работы
Создание оконного проложения позволяющее проводить конвертацию валют согласно курсу ЦБ РФ(приложение не работает напрямую с банковскими счетами, даныне полученые из него носят ознакомительный характер)
# Структура проекта
libs - необходимые библиотеки

natives/cpp - исполняемые файлы, файлы заголовка

src - файлы оконного интерфейса
# Зависимости
CMake - 3.10
С++ - ISO 20
