# Приложение - конвертер валют
В этом репозитории представлена программа, позволяющая проводить конвертации валют.
Приложение имеет оконный интерфейс, возможность выбора цветовой темы(темной/светлой) и языка(русского/английского).
# Цель работы
Создание оконного проложения позволяющее проводить конвертацию валют согласно курсу ЦБ РФ(приложение не работает напрямую с банковскими счетами, даныне полученые из него носят ознакомительный характер)
# Структура проекта
libs - необходимые библиотеки

natives/cpp - исполняемые файлы, файлы заголовка

src - файлы оконного интерфейса
# Зависимости
CMake - 3.10

С++ - ISO 23
# Дополнительный инструментарий в комплекте проекта
Заархивированный установщик необходимой версии CMake

Необходимые библиотеки: CURL, JSON, OpenSDK и JavaFX
# Порядок установки и начала работы
Быстрый вариант: пока в разработке

Стандартный вариант:
  1. Импортировать проект
  
  2. Открыть командную строку PowerShell или любую другую, которая поддерживает многострочные команды
  
  3. Перейти в директорию проекта (к файу CMake)
  
  4. Прописать команды
     3.1.` >>    --cmake -B build`
     3.2.` >>    --cmake --build build --config Release`
     3.3.` & ".\libs\openjdk-25.0.2\bin\java" `
        >>     --module-path ".\libs\javafx-sdk-25.0.2\lib" `
        >>     --add-modules javafx.controls,javafx.fxml `
        >>     --enable-native-access=javafx.graphics `
        >>     -jar converter.jar

При повторном запуске проекта, при наличии .jar файла в директории проекта, необходимо прописывать в командной строке только третью команду.
